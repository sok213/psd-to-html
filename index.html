What is PSD to HTML?

PSD to HTML is a common method for developers to take a PhotoShop file(PSD) and convert that static design into a functional web page using HTML/CSS and most likely a front-end framework like Twitter Bootstrap(used here) or Foundation Framework along with some JavaScript for interactivity.

JavaScript or jQuery is used to add interactive features such as dropdown menus, dynamically adding data to the page, or for mobile responsiveness.

Why PSD to HTML?

Many companies have designers who create web pages mockups.
It is usually the Web Developer's job to code the design into a functional web page.

Knowing how to convert a PSD file to HTML is an essential skill for any Web Developer.
